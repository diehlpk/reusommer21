# Challenges for coupling approaches for classical linear elasticity and bond-based peridynamic models for non-uniform meshes and damage

Simulations for the comparison of the three methods without damage. With these scripts, we compare both models in the case of pure linear elasticity.

## Dependencies

* [numpy](https://numpy.org/)
* [matplotlib](https://matplotlib.org/)
* [ipython](https://ipython.readthedocs.io/en/stable/)
* [notebook](https://jupyter.org/install)


## Usage

```bash
python3 -m venv deps
source deps/bin/activate
pip install -r requirements.txt
ipython notebook
```
